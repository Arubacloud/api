# Inizia

Benvenuto nella documentazione di Cloud Management Platform, la tua soluzione cloud all-in-one. Questo documento fornisce una panoramica delle API esposte e linee guida su come i clienti possono interagire con esse.

## Introduzione

Le API RESTful di Aruba Cloud Platform sono disponibili su HTTPS con JSON come formato dati. Il paradigma utilizza metodi HTTP e codici di stato HTTP per specificare richieste e risposte.

## Documentazione API

### 1. Autenticazione

Aruba Cloud Platform utilizza OAuth 2.0 per l'autenticazione sicura. Ottieni le credenziali API dal portale della piattaforma Web. Puoi trovare i dettagli sull'autenticazione OAuth in [questa sezione](/docs/authentication).

## Utilizzare le API

Per utilizzare le API, segui i seguenti step:

1. **Autenticazione:**
   - Ottieni le credenziali tramite la piattaforma web
   - Usa la collezione Postman di esempio per ottenere il token di autenticazione [here](/docs/authentication)

2. **Gestione Risorse:**
   Utilizza la sezione per interagire con le API
:::info

Per utilizzare le API,usa il seguente FQDN:

https://api.arubacloud.com
:::
